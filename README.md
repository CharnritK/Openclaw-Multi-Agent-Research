# Openclaw Multi Agent Research

Research and planning workspace for a practical multi-agent stack centered on OpenClaw, with focused tracks for Paperclip supervision and Hermes Agent interoperability.

## Current objective

Build a decision-ready research baseline that answers:
- what multi-agent framework patterns actually fit Point's OpenClaw workflow
- where Paperclip should sit in the stack
- how Hermes Agent should integrate without creating brittle overlap
- what to prototype next in the smallest safe steps

## Working rules

- commit at task level
- keep each commit modular
- finish the current bounded task before starting the next one
- follow the roadmap continuously by taking the next best ready item

## Repo structure

- `docs/planning/openclaw-multi-agent-research/` canonical planning bundle
- `research/` source-backed research notes and comparisons
- `notes/` small working notes, gaps, and follow-ups

## Current status

The repo is now being shaped into a planning-first research workspace. The first milestone is to publish the canonical planning bundle, then add framework and integration research baselines.
